**Australian Dream / PumAI** is a SaaS product — AI agents automating sales, support, and marketing — adapted to the Australian market using SMS as the primary channel (via 160.com.au) and GPT-4o Mini as the AI engine.

**Why:** SMS has 94-98% open rate in Australia with universal penetration. 2.5M+ SMEs in AU, 66% already use SMS software.

**How to apply:**
- All UI/content must be in English (Australian market)
- Tech stack: Next.js + TypeScript + Tailwind v4, Node.js/Python backend, PostgreSQL + Redis, 160.com.au SMS API, OpenAI GPT-4o Mini
- Target verticals: Healthcare, Automotive, Real Estate (high priority), E-commerce, Trades, Hospitality (medium)
- Pricing: Starter A$299/mo (+A$500 setup), Growth A$649/mo (+A$900 setup), Enterprise A$1,499+/mo (custom setup)
- Design: Dark theme, violet/purple accents (#8B5CF6), glassmorphism
- IMPORTANT: No references to competitor names anywhere in the product
